Projet Meshcentral

Installation de MeshCentral :
Nous avons besoin de Node.js et npm :
apt install nodejs
apt install npm

Ensuite nous allons créer un répertoire d’installation de MeshCentral :
mkdir meshcentral
cd meshcentral

Puis on passe à l’installation :
npm install meshcentral

MeshCentral est donc installé, premier lancement :
node ./node_modules/meshcentral –cert <nomdedomaine ou ip>
